Setting up your Chalmers email account

IMAP

  • imap.chalmers.se - port: 993
  • username : CID
  • Auth method: CID password
  • Connection: SSL/TLS // Note : Thunderbird test doesn’t work correctly.

SMTP

  • smtp.chalmers.se : 587
  • username : CID
  • Auth method: NTLM (TLS also works)
  • Connection: STARTTLS (if required)

iOS

Detail setting from IT portal - Select Exchange account - On the first page, use ‘CID@chalmers.se’ - Domain: NET - Server: webmail.chalmers.se // Note: if you found some information page from Chalmers with the server name “m.outlook.com”, that is intended for Chalmers student accounts.

Thunderbird/Outlook/Exchange integration

Tools -> addon -> “Lightning”
